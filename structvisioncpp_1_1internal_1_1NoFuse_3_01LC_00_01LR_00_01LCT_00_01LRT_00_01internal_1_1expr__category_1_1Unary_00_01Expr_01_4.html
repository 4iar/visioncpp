inlinestatic

the no_fuse function to execute an expression node with one operand.

It recursively calls the no_fuse function for its RHS; collects the result; launch a device kernel for the current expr with the new collected result; and returns a leafNode representing the output result of the expression.

Parameters
expr: the expression passed to be executed on the device
dev: the selected device for executing the expression
Returns
the leafNode representing the result of the expression execution.
